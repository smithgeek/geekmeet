1. Download the Tinker OS image. You should use the TinkerOS_Debian system.
  2. Download Etcher
  3. Use etcher to flash the OS image to an sd card.
  4. Power on the device. Note: You should have a display connected the first time you boot or you may an error that requires reflashing the sd card.
  5. Download and run the install script.

    6. Option 1: Connect mouse and keyboard

    1. Right click and save start.sh to the desktop.
    2. Right click the file and select properties
    3. Switch to the Permissions tab and change Execute to Anyone
    4. Double click the file and select Execute in Terminal

    Option 2: Using ssh

    1. Connect to the device using ssh linaro@{ip_address} password linaro
    2. Download and run the install script.
